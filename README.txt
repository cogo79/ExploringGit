This project is just my little sandbox for playing and trying out different commands with Git.




Delete a remote branch:
git push origin :myRemoteBranchThatIWantToDelete

Delete a local branch:
git branch -d myBranch

Create a shortcut command for showing a graph of commit history
git config --global alias.lga "log --graph --oneline --all --decorate"
to run this shortcut simply enter "git lga"

Make branch point to previous commit:
git reset a70b980

Fix if Git tracks a file that is in .gitignore file:
git rm --cached file.txt
